# Class Rank: Best X Laps (Consecutives)

![RHFest](https://github.com/arnaudmorin/RotorHazard-Class-Rank-Best-X-Laps-Cumulative/actions/workflows/rhfest.yaml/badge.svg)

A RotorHazard plugin that adds a 'Best X Laps (Consecutives)' class ranking method. This method finds the fastest consecutive laps for each pilot across all races in a class.

### Features

- Configurable number of consecutive laps to rank by
- Works across all heats and races in a class
- Handles staggered starts correctly
- Proper tie handling in rankings

### Usage

1. Install the plugin in your RotorHazard instance
2. In Class settings, select 'Best X Laps (Consecutives)' as the ranking method
3. Configure the number of laps to use for ranking
